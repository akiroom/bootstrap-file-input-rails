# bootstrap file input rails

## About

bootstrap-file-input is made by Gregory Pike.


## Usage

In your Gemfile, add:

```
gem 'jquery-ui-rails'
```

To require bootstrap-file-input, add the following to your application.js:

```
//= require bootstrap-file-input
```
